# Dog Product Sales Data Entry and Cleaning Project

## Project Overview
This project analyzes dog product data in Google Sheets. The goal was to clean and standardize the dataset, then identify patterns in product categories, quantities, and pricing using pivot tables and charts.

## Dataset
The dataset includes:
- Product Name
- Category
- Price
- Quantity
- Supplier
- Date Ordered

The raw data contained inconsistencies in capitalization, formatting, and pricing.

## Data Cleaning
- Standardized capitalization across text fields  
- Formatted prices to currency with two decimal places  
- Handled one missing price by marking it as N/A  
- Improved overall consistency for analysis  

## Data Transformation
- Created Price Range (0–10, 10–20, 20–30, 30+)  
- Created Quantity Range (1–5, 6–10, 11–15, 16+)  
- Standardized categories (e.g., Toy/Chew → Toys)  

## Analysis
- Product Count by Category – identifies the most common product types  
- Quantity Distribution – shows how order sizes are distributed  
- Price Distribution – highlights common price ranges  

## Key Insights
- Toys is the most common category  
- Most quantities fall between 1–10 units  
- Most products are priced in the 0–10 range  

## Tools Used
- Google Sheets  
- Pivot Tables  
- IF formulas  
- Charts  

## Conclusion
This project demonstrates data entry, data cleaning, and basic analysis to organize raw data and identify patterns in a structured and meaningful way.
